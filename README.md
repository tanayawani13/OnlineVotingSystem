# Online Voting System

An Online Voting System developed using Java, JSP, Servlets, Eclipse IDE, and PostgreSQL.

## Features
- User Registration and Login
- Voting System
- Admin Panel
- Vote Result Management
- Category and Nominee Management

## Technologies Used
- Java
- JSP & Servlets
- PostgreSQL
- JDBC
- Eclipse IDE
- HTML/CSS
